Features:

  • Native Android application, written in pure Java and XML
  • Automatic daily wallpaper changes, powered by Firebase and Python
  • Scrollable launcher widget
  • Fullscreen image view
  • Date picker navigation to view images dating back to 1995
  • And much more...
