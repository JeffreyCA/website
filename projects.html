APOD Gallery more_vert

APOD Gallery is an Android application that lets users to explore the images and videos from NASA's Astronomy Picture of the Day (APOD) service.

APOD Gallery close

Features:

  • Native Android application, written in pure Java and XML and uses elements of Material Design
  • Uses NASA's Astronomy Picture of the Day (APOD) API to retrieve image data and descriptions dating back all the way to 1995
  • Automates phone wallpaper changes daily, powered by Firebase and a Python script which updates the database every time there is a new image
  • Scrollable launcher widget to browse through saved images
  • Fullscreen image view to pan and zoom content
  • Intuitive controls such as: swipe to navigate between pictures, slideable view to bring up descriptions, and date picker to refine selection
